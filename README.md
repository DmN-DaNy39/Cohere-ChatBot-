# Cohere Chatbot
This uses the <a href="https://cohere.com/">Cohere</a> API, the pyttsx3 library and the python speech recognition library.

# How to Use
To use this program use have to have pyttsx3 and speech recognition installed so you could;
```bash
pip install pyttsx3
pip install SpeechRecognition
```
And you have to put in a API key for cohere, to get one you have to make an account on <a href="https://cohere.com/">Cohere's Webite</a> and get one.
```python
co = cohere.ClientV2("#ADD COHERE API KEY :)")
```
Then open the python file.
